# RVVR Checklist

A real-time checklist application for church services.

## Features

- **Real-time updates** - All connected clients see task changes instantly via WebSocket
- **Team-based organization** - Organize checklists by teams (Worship, Tech, Welcome, etc.)
- **Custom sections** - Create time-based sections (pre-service, during-service, post-service) ,
- **Public access** - No login required to view or mark tasks during service
- **Admin management** - Authenticated users can manage teams, sections, and tasks

## Goal

Enable church service team members to track what tasks have been completed and what remains pending, with everyone seeing updates in real-time.
