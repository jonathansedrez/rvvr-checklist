# Sound Setup

Step-by-step instructions for configuring the full PA system from scratch.

## Power Sequence

Always power up in this order to protect speakers from transient pops:

1. **Source gear** — mixing console, DI boxes, rack processors
2. **Signal processing** — compressors, gates, EQ inserts
3. **Power amplifiers** — main PA last

Power down in reverse order at the end of the night.

## Gain Structure

Set gain structure from source to destination:

- Input gain: aim for signal peaking at −18 dBFS on loud passages
- Channel faders: nominal at 0 dB (unity)
- Group/VCA faders: unity
- Master output: leave headroom for the limiter

## FOH Mix Reference Points

| Source | Target level (pre-fader) |
|--------|--------------------------|
| Kick   | −12 dBFS                |
| Snare  | −15 dBFS                |
| Lead vocal | −18 dBFS to −12 dBFS |
| Acoustic guitar | −20 dBFS          |

## Common Issues

**Feedback during soundcheck**
- Identify the ringing frequency using narrow EQ sweeps
- Cut 3–6 dB at the offending frequency before increasing gain

**Hum or buzz on a channel**
- Check for ground loop — try a DI with ground lift
- Inspect cable for physical damage at connectors

**No signal on a channel**
- Verify phantom power if the mic requires it
- Swap cable and retest before replacing the mic
